# Central Bank of Learnable

A basic banking API service with the following features:

### Users can :

- Login
- Deposit money
- Withdraw money
- Transfer funds to other users
- See a list of their transactions
- See their wallet
- See their profile

### Admin can :

- Get all users
- Get single user
- Add users
- Delete users
- Reverse transactions(transfer)
